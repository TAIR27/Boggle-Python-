
# Boggle Word Game

## Description
A word search game developed in Python with a graphical user interface using Tkinter.

## Features
- DFS-based search for valid word paths on a 2D board  
- Word validation using a dictionary  
- Scoring system based on path length  
- Interactive GUI with timer and real-time feedback  

## Technologies
- Python  
- Tkinter  

## How to Run
1. Run the main Python file  
2. Start the game and find words within the time limit
